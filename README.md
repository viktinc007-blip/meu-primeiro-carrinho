# Carrinho de Compras Simples

Projeto simples feito com HTML, CSS e JavaScript.  
Feito para praticar conceitos básicos de programação web e lógica de um carrinho de compras.

## Funcionalidades
- Adicionar e remover produtos
- Mostrar quantidade de itens e preço total

## Como usar
Abra o arquivo `index.html` no navegador e experimente adicionar ou remover produtos.
